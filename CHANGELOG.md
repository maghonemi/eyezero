# Changelog

All notable changes to EyeZero will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-05

### Added
- Initial release of EyeZero
- Hand gesture tracking using MediaPipe
- Real-time cursor control via hand movements
- Pinch gesture for clicking
- Visual cursor indicator
- macOS DMG installer
- Camera and accessibility permission handling
- Smooth cursor movement with throttling
- AppleScript fallback for mouse control
- Comprehensive documentation

### Features
- Hand tracking with MediaPipe
- System-wide mouse control
- Click detection via pinch gesture
- Cursor visualization overlay
- Window controls (minimize, close)
- Dock icon support
- Universal build (Intel & Apple Silicon)
